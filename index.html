• Task 1: Validate an Email Address Create a regular expression that checks if a given string is a valid email address. The basic rules are: It must contain an @ symbol. The domain part must have at least one . (dot) after the @. The domain part should only contain alphanumeric characters and dots (e.g., gmail.com or yahoo.co.uk).



  • Task 3: Validate a Password Create a regular expression to validate a password based on the following criteria: It should be at least 8 characters long. It should contain at least one lowercase letter, one uppercase letter, one digit, and one special character.
